Generasi 'menolak' tua.

Burung merpati
jadi 'kurir' surat

Cuma burung merpati yang cocok dengan tugas ini. Sebagai 'kurir' dadakan, mereka dilatih secara terus menerus untuk menemukan jalan pulang ke rumahnya. Nggak heran kalau para militer dan intel pakai merpati pos ini untuk mengirimkan surat rahasia.

Obor dan lampu teplok
jadi primadona

Dulu, untuk ke luar rumah aja harus bawa obor. Kalau di rumah, justru lampu teplok yang jadi primadonanya. Menjelang malam, mereka harus mengisinya dengan minyak tanah supaya tetap menyala sepanjang malam.

Sejauh apa pun,
sepeda onthel masih
bisa diandalkan

Sejauh apa pun jaraknya, selama ada sepeda onthel pasti aman deh. Memang capek, tapi ini gratis dan tanpa bahan bakar. Namanya dulu bukan klakson, tapi bel sepeda dengan suara khasnya 'kring-kring'. Lucunya lagi, kamu harus merelakan sandalmu sebagai remnya.

Membuat api dengan batu

Kalau sekarang, membuat api dengan batu hanyalah cara darurat saja. Padahal dulu, ini adalah satu-satunya cara untuk membuat sumber api. Setidaknya kita harus menggunakan batu berbentuk lancip dan menggesekkannya terus hingga menimbulkan percikan api.

Coba lihat sekarang, semua serba instan. Bersyukurlah kamu yang hidup di abad 21! Mau menyalakan api untuk memasak, sudah ada gas, mau pergi jauh pakai mobil, sudah ada bahan bakar.
